BRAND GUIDELINE WEBSITE

ROLE/ DEVELOPER ■  CONTEXT/ COURSE PROJECT ■  PLATFORM/ DESKTOP&MOBILE ■  DATE/ JULY 2017
HTML   CSS   PHOTOSHOP   AFTER EFFECT  

This was my course project with objectives to create an responsive online site across multiple platforms that defined and adhered to brand guidelines. The pupose of this website is for third parties to have a clear set of guidelines which may include the brand's history, vision, personality, colour palette, type style, logo use, etc. This website is also my trial on implementing JavaScripts plugins.

 

THE PROCESS

CONCEPT:   I choose MailChimp for this project. The idea is to keep the page clean in order to make the information of the guidelines readable. I made some sketches to see the layout and the flow of the user experience when they are opening it.

PREPARATION:   To add more fun site to the plain white backgound, I prepare an image background, inspired by ke icon, the monkey name Fredie, I created an ornament image using banana leaves, envelope and some graphics. To enriched the content of the website I created a welcoming video in After Effect.

 

PRODUCTION:   The Layout is build in HTML, CSS and using JavaScript plugin for the scrolling effect of the video and other information. The Social Media buttons are made with the sprite technique to keep the performance of the website to load quickly.

Disclaimer: This website is an assigment for learning purposes and is not the official brand's website.
