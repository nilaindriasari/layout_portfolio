WORDPRESS ARTICLE WEBSITE

ROLE/ DEVELOPER ■  CONTEXT/ COURSE PROJECT ■  PLATFORM/ DESKTOP&MOBILE ■  DATE/ AUGUST 2017
WORDPRESS   CSS   JAVASCRIPT   PHOTOSHOP   ILLUSTRATOR  

This course project objective is to create a responsive articles site across multiple platforms driven by a Content Management System (CMS) website, WordPress. The customization of website was done through designing a basic template from scratch in HTML and CSS then implement in the CMS to create an article of interest.

 

THE PROCESS

CONCEPT:   I did a preliminary sketch of the website to get a good idea of its visual and implement it in Illustrator. Bright and fun colors were chosen to match the content of fun articles .


GRAPHIC:   I created the logo in Illustrator and decided to make a gif animation to enhanced the fun factor to the website. I prepared the social media sprite and other icons in Illustrator, while the full size and thumbnails images for the articles created and retouched in Photoshop using free licence images.

FRONT END:   The main programming language used to build this website is PHP to fully utilize WordPress, although the styling is still done in CSS. JavaScript plugins is used for the markup words in the article's content. This website also incorporated some widgets and plugin from WordPress.
