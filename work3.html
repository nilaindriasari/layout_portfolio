N.G.O. LANDING PAGE

ROLE/ DEVELOPER ■  CONTEXT/ COURSE PROJECT ■  PLATFORM/ DESKTOP&MOBILE ■  DATE/ JULY 2017
HTML   CSS   JAVASCRIPT   PHOTOSHOP   ILLUSTRATOR   AFTER EFFECT   SASS  

This course project objective is to create an responsive landing page for N.G.O. across multiple platforms. This website is using a CSS preprocessing language (Sass) to improve the workflow and also JavaScript for the parallax effect.

 

THE PROCESS

CONCEPT:   I picked Red Pencil as the client for this course project and made sketches for the main website layout. The colour palette that they are using mostly is red. The combination of red, white and black picked to augment the dramatic effect.

GRAPHIC:   I wanted to create an image of a child that can be animated on hover. Therefore, I prepare the animation layers on Photoshop and work on digital imaging process from downloaded free licensed images. The drawing image on the paper that the child hold was done in After Effect. To enhance the visual, some brush strokes are created for the titles by Illustrator and I prepared the logo, retouched client's photos in Photoshop.

   

FRONT END:   The Layout is built in HTML and CSS using Sass. The animated counter number, parallax effect and modal page for the donation form are built using JavaScript. The child's hand hover animation is done in CSS.

Disclaimer: This website is an assigment for learning purposes and is not the official brand's website.
