CUISINE WEBSITE

ROLE/ DEVELOPER ■  CONTEXT/ COURSE & SIDE PROJECT ■  PLATFORM/ DESKTOP&MOBILE ■  DATE/ AUGUST 2017
HTML   CSS   JAVASCRIPT   PHOTOSHOP   ILLUSTRATOR  

This course project objectives is to create a responsive online resume across multiple platforms for a cuisine company. This website has JavaScript integrated for some of its functions.

 

THE PROCESS

CONCEPT:   This is course project and also currently ongoing side project with real client. The client planned to have a website for her cuisine business. I created a mockup in Illustrator to give her an idea for website's visual. The first thing to do is creating the logo for the business. The main logo and the theme colour was decided between some alternatives.

 

GRAPHIC:   Since the real photos of the products are yet unavailable, I used products photos from free licence images and did some digital retouched in Photoshop, so were other graphic elements, such as retouched background, map container, social media icons, etc.

FRONT END:   The Layout is built in HTML, CSS with some JavaScript. Each of the product pages were toggled and built using JavaScript. The green hover tooltips on each products created in CSS. The Google Map styling and Twitter live tweet were also done with some JavaScript API code
